# BeyondPaste
Simple self hosted Pastebin-like tool allowing you to have the pastes uploaded on your own server.  
Running on PHP7, it is using Bootstrap 4 for its frontend, Geshi for the syntax highlighting.

# Features
* Paste upload
* Syntax Highlighting
* Unlisted Pastes
* One Time Read Paste
* Paste auto remove
